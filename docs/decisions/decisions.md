# Decisions

For the creation of this starter kit we are working hard to ensure that the ways that everything works is established using a few important principles:
- Clean Code Principles
  - Verbose, meaningful names
  - Reduce redundancy
  - Reduce side effects
  - Small functions
  - Etc...
- Easy to find resources and support for external packages
- Well maintained packages
- Keep boilerplate to a minimum
